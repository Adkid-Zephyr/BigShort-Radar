# INDICATORS

> 这是系统的"翻译表"。指标定义、计算口径、阈值、历史含义都在这里。
> **翻译卡（含义/误判/历史案例）由用户手写，模型不能编造。** 模型补不了的写"待用户补"。

## 模板（每个新指标按这个写）

```
### <指标名>
- **代码**：在系统里的 name（如 yield_curve_10y2y）
- **数据源**：FRED:T10Y2Y / YF:^VIX / 其他 URL
- **更新频率**：日 / 周 / 月
- **计算口径**：原始序列直接用 / 衍生公式
- **阈值**（方向：↑ 升=风险升 还是 ↓ 降=风险升）
  - GREEN：< x
  - YELLOW：x – y
  - RED：> y
- **阈值依据**：历史分位 / 文献 / 经验值
- **翻译卡**（用户手写）：
  - 这个指标到 RED 历史上意味着什么
  - 最容易被误读的情况
  - 与哪些指标交叉看
- **最近一次校准**：YYYY-MM-DD
```

### vix_term_structure
- **代码**：vix_term_structure
- **数据源**：YF:^VIX / YF:^VIX3M（取比值）
- **更新频率**：日
- **计算口径**：value = VIX_close / VIX3M_close（按交易日对齐）
- **阈值**（方向：↑ 升=风险升）
  - GREEN：< 0.95（contango，平静）
  - YELLOW：0.95 – 1.0（紧张）
